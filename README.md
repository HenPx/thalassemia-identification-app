### Manual Installation

- Clone the repository:

```bash
git clone https://github.com/HenPx/thalassemia-identification-app.git
```

- Install the requirements:

- Create .env file 

```bash
python -m venv
```

- Active .env file (active before run app)

```bash
.venv/Scripts/activate
```

- Install all requirements for python (for local)

```bash
pip install -r all_requirements.txt
```

- Run the application:

```bash
streamlit run app.py
```
